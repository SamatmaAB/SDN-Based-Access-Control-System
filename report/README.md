# Report Assets

Use this folder for submission material.

- `report.md`: report-ready write-up
- `screenshots/`: optional screenshots of terminal output, Mininet CLI, or `ovs-ofctl dump-flows`

Suggested screenshots:

- regression test output
- successful `verify_access.py` run
- Mininet CLI showing one allowed and one blocked ping
- `sudo ovs-ofctl -O OpenFlow13 dump-flows s1`
